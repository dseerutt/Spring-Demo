Bienvenue dans l'application Spring Demo !

Voici la liste des endpoints :

Index :

  • GET the index page: localhost:%PORT%

Client :

Computer :

  • GET all computers: localhost:%PORT%/client/computer
  • GET a computer: localhost:%PORT%/client/computer/{id}

Sale :

  • POST a sale: localhost:%PORT%/client/sale
  • GET a sale: localhost:%PORT%/client/sale/{id}

Provider :

Computer :

  • POST a computer: localhost:%PORT%/provider/computer
  • PUT a computer: localhost:%PORT%/provider/computer/{id}

Computer provisioning :

  • POST provision computers: localhost:%PORT%/provider/computer/provision
  • POST deprovision computers: localhost:%PORT%/provider/computer/deprovision

Computer status update:

  • PATCH computer status: localhost:%PORT%/provider/computer/{{id}}/status

Sale :

  • GET all sales: localhost:%PORT%/client/sale
  • PUT a sale: localhost:%PORT%/provider/sale/{id}
  • DELETE a sale: localhost:%PORT%/provider/sale/{id}
