Bienvenue dans l'application Spring Demo !

Voici la liste des endpoints :

Index

  • GET the index page: localhost:%PORT%

Computer

  • POST a computer: localhost:%PORT%/computer
  • GET all computers: localhost:%PORT%/computer
  • GET a computer: localhost:%PORT%/computer/{id}
  • PUT a computer: localhost:%PORT%/computer/{id}
  • Computer deletion is not available

Sale

  • POST a sale: localhost:%PORT%/sale
  • GET all sales: localhost:%PORT%/sale
  • GET a sale: localhost:%PORT%/sale/{id}
  • PUT a sale: localhost:%PORT%/sale/{id}
  • DELETE a sale: localhost:%PORT%/sale/{id}
